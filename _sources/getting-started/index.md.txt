## このセクションで学ぶこと

ロボットプログラミングの冒険を始める前に、必要な道具（ソフトウェア）を準備します。RPGゲームで最初に装備を整えるのと同じですね！

## 環境の準備

### 💻 あなたのPCはどれですか？

#### 🐧 **Linuxユーザー（Ubuntu）**
- Ubuntu 20.04 LTS がインストールされている
- → **このページの「ソフトウェアのインストール」に進む**

#### 🪟 **Windowsユーザー**
- Windows 10 または Windows 11 を使用
- → **まず [WSLでLinux環境を手に入れよう](wsl-setup.md) を完了させる**
- → その後、このページの「ソフトウェアのインストール」に進む

#### 🍎 **Macユーザー**
- 申し訳ありませんが、本演習はLinux環境が必要です
- → **[WSLでLinux環境を手に入れよう](wsl-setup.md) を参考に、仮想環境を構築してください**

### 📊 システム要件
- **メモリ**: 4GB以上推奨
- **ストレージ**: 20GB以上の空き容量
- **ネットワーク**: Wi-Fi接続とインターネット接続

## ソフトウェアのインストール

### ステップ1: ターミナルを開く

まずは「ターミナル」というツールを開きます。

#### ターミナルって何？🎬
映画でハッカーが使っている、あの黒い画面に緑の文字が流れるかっこいいやつです！実際は、マウスでクリックする代わりに、キーボードでコマンド（命令）を入力してコンピュータを操作するツールです。

なぜターミナルを使うの？
- 🚀 マウス操作より速い！
- 💪 プロっぽくてかっこいい！
- 🤖 ロボット制御には必須！

1. キーボードで `Ctrl + Alt + T` を押す
2. または、画面左下の「アプリケーションを表示」から「端末」または「Terminal」を探してクリック

黒い画面（または紫の画面）が出てきたら成功です！君もこれでハッカー気分！

### ステップ2: システムを最新の状態にする

```bash
# パッケージリストを更新（最新の情報を取得）
sudo apt update

# インストール済みのソフトウェアを更新
sudo apt upgrade -y
```

💡 **豆知識**: `sudo`は「スーパーユーザー権限で実行」という意味。管理者として命令を実行するときに使います。スマホでアプリをインストールする時に「許可しますか？」って聞かれるのと同じです！

### ステップ3: 基本ツールのインストール

```bash
# プログラミングに必要な基本ツールをインストール
sudo apt install -y \
    build-essential \
    git \
    curl \
    wget \
    python3-pip \
    python3-venv
```

### ステップ4: ROSのインストール準備

ROSは「Robot Operating System」の略で、ロボットを動かすための特別なシステムです。

```bash
# ROSのインストールに必要なツール
sudo apt install -y python3-vcstool python3-catkin-tools
```

## ワークスペースの作成

プログラムを整理して保存する場所（ワークスペース）を作ります。

```bash
# ホームディレクトリに移動
cd ~

# ROSワークスペース用のフォルダを作成
mkdir -p ~/ros/enshu_ws/src

# 作成したフォルダに移動
cd ~/ros/enshu_ws/src
```

## エディタの準備

プログラムを書くためのエディタを準備します。初心者には**Visual Studio Code**がおすすめです。

```bash
# Visual Studio Codeのインストール
sudo snap install code --classic
```

## 確認クエスト 🎯

以下のコマンドを実行して、インストールが成功したか確認しましょう：

```bash
# Pythonのバージョン確認
python3 --version

# Gitのバージョン確認  
git --version

# 現在のディレクトリ確認
pwd
```

すべてのコマンドでバージョン番号や現在地が表示されれば、準備完了です！

## トラブルシューティング 🔧

### Q: 「コマンドが見つかりません」と表示される
A: スペルミスがないか確認し、`sudo apt update`を実行してから再度試してください。

### Q: パスワードを入力しても画面に表示されない
A: これは正常な動作です。セキュリティのため、パスワードは画面に表示されません。正しく入力してEnterキーを押してください。

### Q: エラーが出て進めない
A: 慌てずに！エラーメッセージを読んで、先生やTAに相談しましょう。
